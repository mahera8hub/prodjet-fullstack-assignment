PRODJET Full Stack Assignment

A waitlist registration app where users sign up as Student, Brand, or College and gets a waitlist number and referral code.

The Tech Stack I used:

- Frontend: React + Vite, plain CSS
- Backend: Node.js + Express
- Storage: JSON file

How to Run:

1.Backend


cd backend
npm install
node server.js


Runs on `http://localhost:5000`

2. Frontend

bash
cd frontend
npm install
npm run dev


Runs on `http://localhost:3000`

API:

POST /waitlist/register
